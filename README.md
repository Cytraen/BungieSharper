# BungieSharp

## About
**BungieSharp** is a .NET client library for the Bungie.net API.  
**BungieSharp.Generator** consumes [Bungie's OpenAPI spec,](https://github.com/Bungie-net/api) and turns it into code usable by BungieSharp.
